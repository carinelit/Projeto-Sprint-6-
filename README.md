# Projeto Integrado - Loja Online Ice

## Objetivo 
- Identificar padrões que determinam se um jogo tem sucesso
- Analisar tendências de plataformas e gêneros por região
- Fornecer recomendações para campanhas publicitárias

## Dataset 
Projeto Bootcamp - Triple ten - Sprint 6

## Perguntas para Analíse 
Quais possíveis sucessos ?
Com deve atuar as campanhas publicitarias do próximo ano ?
Quais gêneros Priorizar?

## Ferramentas 
- Python
- Pandas
- Matplotlib
- Seaborn
- Grafícos
- Spicpy 
- Math 
- Numpy 

## Conclusões 
Foco em PS3 e X360 para máximo alcance.
Trabalhar campanhas publicitarias especiais no Japão, podemos notar fortes preferencias por portáteis e role-playing. 
Priorizar gêneros como : Action, Sports e Shooter.
